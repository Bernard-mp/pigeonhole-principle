Common Misconceptions


1. Assumption that atleast one pigeon is present in every pigeonhole.
2. It does not say which piegeon is present.
3. It does not say which piegeonhole has 2 or more pigeons in basic or minimum of how many in extended.

Illustration

Here if we consider the colors as the pigeons and the boxes as the pigeonholes then according to the pigeonhole principle:
1. Basic principle states that there are atleast 2 or more colors in atleast 1 pigeonhole.
2. Extended principle states that there is atleast 3 or more colors in atleast 1 piegeonhole.
This holds true for any ways of selection or combination where in each and every combination there will be atleast one container having 3 or more colors.

            Given:

             n = number of colors = 5

             m = number of containers = 2
            
            Using Formula: 

            ⌈n / m⌉ or ⌊(n − 1) / m⌋ + 1

            ⌈5 / 2⌉ or ⌊(5 − 1) / 2⌋ + 1 

            ⌈2.5⌉ or  ⌊2⌋ + 1 = 3
