Pigeonhole Principle

Overview


The pigeonhole principle (also sometimes called the Dirichlet drawer principle) is a simple yet powerful idea in mathematics that can be used to show some surprising things, as we’ll see later. In this article, we’ll first define what the pigeonhole principle is, followed by some examples to illustrate how it can be applied. Then we’ll discuss the generalized version of the principle and give several examples that use the generalized version. Finally, we’ll show some particularly clever applications that make use of the pigeonhole principle.


The Basic Pigeonhole Principle


In this section, we’ll first define the basic pigeonhole principle and prove it. Then we’ll show three examples that use this basic principle..

Generalized pigeonhole principle:
1. If n pigeons occupy m pigeon holes and n > m then there exists atleast one pigeon hole that will have atleast 2 pigeons.
2. If m > n i.e., number of pigeon holes is greater than the number of pigeons then it is a contradiction.





Some Examples of Pigeonhole Principle


1. Two or more people reading this blog will have the same birthday.
2. If you pick five cards from a standard deck of 52 cards, then at least two will be of the same suit.

Extended Pigeonhole Principle


It states that if n pigeons are occupy m pigeonholes (The number of pigeons is very large than the number of pigeonholes), then atleast one of the pigeonholes must contain
